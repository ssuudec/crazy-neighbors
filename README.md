# Crazy Neighbors

Canlı Oyun Linki: https://ssuudec.itch.io/crazy-neighbors

## Proje Hakkında

Crazy Neighbors, Türk mahalle atmosferine sahip, sıra tabanlı ve komik bir 2D atış oyunudur.  
Oyuncular kalkan açma, çift atış ve moral bozma gibi aksiyonları kullanarak rakibini yenmeye çalışır.

## Kontroller

- Fare ile nişan al
- Sol tık ile atış yap
- Aksiyon butonları ile:
  - Kalkan aç
  - Çift atış yap
  - Rakibin moralini boz

## Geliştirme Ortamı

- Unity 6 (WebGL build)
- C#
